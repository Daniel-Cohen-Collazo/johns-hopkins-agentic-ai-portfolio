# Johns Hopkins Agentic AI Portfolio

A project portfolio built through the **Johns Hopkins Agentic AI Certificate Program**, focused on applied AI systems, Retrieval-Augmented Generation (RAG), vector databases, tool-calling workflows, and business-oriented AI use cases.

This repository documents the progression of my work across the certificate, with each project building on the previous one and expanding my hands-on experience with agentic systems.

## Portfolio structure

### Project 1 — DualLens Analytics
A dual-lens analytics project that combines financial performance data with qualitative AI initiative analysis to help evaluate organizations from both a market and innovation perspective.

### Project 2 — Autonomous Financial Analyst
An AI-powered financial research agent that combines stock data, financial news, sentiment analysis, and private-document retrieval into grounded, decision-ready research outputs.
