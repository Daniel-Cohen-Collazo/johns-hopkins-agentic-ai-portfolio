# Project 1 — DualLens Analytics

DualLens Analytics is a financial and innovation analysis project built as part of the Johns Hopkins Agentic AI Certificate Program.

## Overview

This project applies a dual-lens framework to investment analysis by combining:

1. **Quantitative financial performance data**
2. **Qualitative analysis of AI initiatives and strategic documents**

The goal is to create a more complete picture of organizational potential by evaluating both current market performance and future-facing innovation signals.

## What it does

- Analyzes financial metrics across companies
- Ingests and processes PDF-based AI initiative documents
- Uses chunking and vector retrieval to query strategic information
- Combines structured financial and qualitative data into a unified analysis workflow

## Skills demonstrated

- Retrieval-Augmented Generation (RAG)
- PDF ingestion and chunking
- Vector database workflows
- AI-assisted business analysis
- Structured reasoning with mixed data sources
