# Project 2 — Autonomous Financial Analyst

Autonomous Financial Analyst is an AI-powered financial research agent built as part of the Johns Hopkins Agentic AI Certificate Program.

## Overview

This project expands on earlier work by introducing an agent-based workflow that combines multiple research tools into a single financial analysis process.

The system is designed to reduce the time required to gather, compare, and synthesize market data, financial news, sentiment signals, and private-document context.

## What it does

- Pulls stock price and market data
- Analyzes historical stock performance
- Searches financial news
- Performs sentiment analysis
- Queries private files using RAG
- Produces grounded financial research outputs

## Technologies used

- Python
- LangGraph
- OpenAI API
- ChromaDB
- yfinance
- Tavily Search API
- PyPDF
- tiktoken

## Skills demonstrated

- LangGraph agent design
- Tool orchestration
- Retrieval-Augmented Generation (RAG)
- Vector database retrieval
- Financial data analysis
- Business-focused AI workflow design
